# Poetpack_tmpl
##### - About
A template for python package with poetry



##### How to 

###### 1. Prepare the envirment
```bash
python -m venv venv
source venv/bin/activate
```

###### 2. Install requirement
```bash
pip install -r requirements.txt
poetry install 
```


###### 3. Run
```bash
poetry run poetpack_tmpl
```
>>> or
```bash
poetry run poetpack_tmpl --help
```

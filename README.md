# Filecheck_pkg
##### - About
A python package for file integrity check.



##### How to 

###### 1. Prepare the envirment
```bash
python -m venv venv
source venv/bin/activate
```

###### 2. Install requirement
```bash
pip install -r requirements.txt
poetry install 
```


###### 3. Run
```bash
poetry run filecheck_pkg
```
>>> or
```bash
poetry run filecheck_pkg --help
```
